# ⚡ FlashFlash - Rapid Learning 10-Foot UI Flashcards

**FlashFlash** es una aplicación web de una sola página (SPA) diseñada para la memorización matutina ágil de alto rendimiento mediante pantallas grandes y Smart TVs (**10-Foot UI**).

---

## 🌟 Características Principales

- **Diseño de 10 Pies (10-Foot UI)**: Interfaz con contraste ultra alto (`16:1`) en tono oscuro `#121212`, pensada para la máxima nitidez y prevención de fatiga visual a distancia.
- **Control Multimodal**: Compatible con teclados inalámbricos y mandos a distancia (`Espacio` para pausar/reproducir, `Flecha Izquierda/Derecha` para cambiar de tarjeta, `Escape` para salir).
- **Higienización Inteligente de Texto**: Sanea automáticamente entradas sucias con espacios redundantes, saltos de línea irregulares o comas múltiples (Heurística #5 de Nielsen).
- **Auto-Pausa por Pérdida de Foco**: Pausa automáticamente la proyección si cambias de pestaña o si el dispositivo entra en reposo.
- **HUD con Auto-ocultamiento**: Los botones táctiles/remotos de `64px` se ocultan a los 3 segundos de inactividad del cursor para una experiencia de TV cinematográfica.

---

## 🎨 Estándares de Diseño Implementados

- **10 Heurísticas de Usabilidad de Jakob Nielsen** (#1, #3, #5, #9)
- **Principios de Accesibilidad MOSIP**
- **Manual de Experiencia UX4G** (Tipografía 'Noto Sans' y accesibilidad universal)
